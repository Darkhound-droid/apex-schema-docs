# Usage Guide

## 1) Open the app

- Page 1 introduces the project and links to generation.

## 2) Select tables (Page 2)

- Confirm schema.
- Search tables using the search field.
- Use checkbox selections to include/exclude tables.
- Pick output format:
  - Markdown
  - JSON
  - Plain Text

## 3) Generate documentation

- Click **Generate Documentation**.
- The app sends selected table names and format to the output page.

## 4) Review output (Page 3)

- Output is generated by `APEX_SCHEMA_DOCS_PKG`.
- Review token estimate.
- Use **Copy to Clipboard** for direct LLM prompt usage.
- Use **Download File** to save `.md`, `.json`, or `.txt`.

## 5) About page (Page 4)

- Contains project background, version details, and V2 roadmap.
